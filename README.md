# SimpleCppDictionary
This project is a simple dictionary api.

# Used
```
GET http://192.168.10.121:18080?word=appl
```
```
[
    "apple",
]
```

## Requirements
C++ compiler with at least C++11 support.
boost library & development headers (1.64 or later).

## build
This wil only build 
```
make build
```

## Run
This wil build and run

```
make run
```
